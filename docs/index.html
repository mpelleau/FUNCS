FunCS

A toolbox for fun scientific dissemination in Computer Science

View on GitHub Download .zip

Description

On this website, you will find a collection of free teaching material that teaches Computer Science through puzzles that were created in the FabLab@UCA. It follows the idea of CS unplugged.
